# 백준 25083 새싹
## 문제
아래 예제와 같이 새싹을 출력하시오
## 예제 출력 1
```
         ,r'"7
r`-_   ,'  ,/
 \. ". L_r'
   `~\/
      |
      |
```
  
## 분석
쉬운 문제이지만, 파이썬의 경우 "와 '를 함께 출력하는 방법을 알아야할꺼 같다.  
\를 이용해 해결했다.



```python
print("         ,r\'\"7")
print("r`-_   ,'  ,/")
print(" \. \". L_r\'")
print("   `~\/")
print("      |")
print("      |")
```
